# 🐺 Wolf-Man

팩맨에서 영감을 받은 미로 아케이드 게임입니다.
Wolf-Man을 조종해 미로의 모든 점을 먹고, 유령들을 피해 살아남으세요!

🎮 **[지금 바로 플레이하기](https://wolf0605.github.io/wolf-man/)**

---

## 게임 방법

미로 안을 돌아다니며 **작은 점(●)** 을 모두 먹으면 스테이지 클리어!
유령에게 닿으면 목숨을 잃고, 목숨이 모두 소진되면 게임 오버입니다.

### 조작법

| 키 | 동작 |
|----|------|
| ⬆️ 방향키 위 | 위로 이동 |
| ⬇️ 방향키 아래 | 아래로 이동 |
| ⬅️ 방향키 왼쪽 | 왼쪽으로 이동 |
| ➡️ 방향키 오른쪽 | 오른쪽으로 이동 |
| `R` | 게임 재시작 |

> 모바일에서는 화면을 **스와이프**하여 이동할 수 있습니다.

## 아이템

| 아이템 | 점수 | 효과 |
|--------|------|------|
| 작은 점 ● | +10점 | 기본 아이템 |
| 파워 펠릿 ✨ | +50점 | 유령을 일시적으로 겁쟁이 상태로 만듦 |
| 겁쟁이 유령 👻 | +200점 | 파워 펠릿 획득 후 유령에게 닿으면 먹을 수 있음 |

---

## 유령

총 **8마리**의 유령이 미로를 돌아다닙니다.

- **평소**: 랜덤하게 이동하며 Wolf-Man을 추격합니다.
- **파워 모드**: 파란색으로 변하며 Wolf-Man을 피해 도망칩니다.
- **점멸 경고**: 파워 모드 종료 직전 유령이 깜빡입니다.

## 승리 / 패배 조건

- **WIN** 🎉 : 미로의 모든 점을 먹으면 클리어!
- **GAME OVER** 💀 : 목숨(❤️ x3)을 모두 잃으면 종료.
